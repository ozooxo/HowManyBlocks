================
How Many Blocks?
================

Challenge the limit of your 3D imagination.

Install
=======

To compile "How Many Blocks?" you will need:

- Python 2.7
- Pyglet

Ubuntu Linux
------------

$ sudo apt-get install python-pyglet
$ python blocks.py
